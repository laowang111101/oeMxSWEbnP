# Python Django图书信息管理系统 python325

## 项目概述

本项目是基于Python Django框架开发的图书信息管理系统，旨在提供便捷的图书数据管理功能。系统使用sqlite3作为数据库，确保数据存储的安全与高效。

## 技术栈

- 后端框架：Python Django
- 数据库：sqlite3

## 功能模块

- 图书信息管理
- 图书检索
- 用户权限控制

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.x
- Django 2.x
- sqlite3

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 初始化数据库
4. 运行系统
